# short instruction
1. `npm i`
2. `credentials.json`
    ```
        "client_id": "<your-client-id>",
        "client_secret": "<your-client-secret>",
    ```
3. `node .`
4. follow the link
5. copy `code` param from url after redirect to localhost
6. paste in console
7. `node .`
8. check the file `result.json` created
